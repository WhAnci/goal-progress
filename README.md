# Goal-Progress
Notion에 임베드하여 사용할 수 있는 심플하고 깔끔한 목표 진행도(Progress Bar) 위젯입니다. 
URL 파라미터 수정해서 목표 이름, 진행률, 다크/라이트 모드를 자유롭게 커스텀할 수 있습니다.

## Parameters

| Parameter | Description | Example | Default |
| :--- | :--- | :--- | :--- |
| **`goal`** (또는 `title`) | 화면에 표시될 목표 이름 | `?goal=다이어트` | 목표 |
| **`progress`** (또는 `value`) | 현재 진행률 (0 ~ 100 숫자) | `&progress=30` | 0 |
| **`theme`** | 강제 테마 설정 (`dark` 또는 `light`) | `&theme=dark` | (시스템 테마 자동 감지) |

## Usage
노션 페이지에서 `/임베드` (또는 `/embed`) 블록을 생성한 뒤, 아래와 같이 주소를 조합하여 붙여넣으세요.

### 1. 기본 사용 (기기 테마 자동 감지)
이름과 퍼센트만 입력하면, 노션 테마(다크/라이트)에 맞춰 자동으로 색상이 변합니다.
```text
https://(내호스팅주소)/?goal=독서하기
```

### 2. 다크모드 고정
theme=dark를 추가해 항상 다크모드로 보이게 설정합니다.

```text
https://(내호스팅주소)/?goal=매일 운동하기&theme=dark
```

### 3. 100% 완료 효과 (폭죽 🎉)
진행도가 100이 되면 게이지가 녹색으로 변하며 폭죽 애니메이션이 터집니다.
